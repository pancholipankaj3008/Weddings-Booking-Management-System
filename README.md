# 📸 Wedding Booking Management System

A comprehensive full-stack MERN Wedding Booking Management System that streamlines photography service bookings, event management, photographer assignment, client communication, and business operations through an advanced admin dashboard.

---

# 📌 Overview

Wedding Booking Management System is a full-stack MERN application developed for a real client to simplify wedding photography booking and business management. The project was built collaboratively in a team environment, where I contributed to both frontend and backend development, implementing core business features, REST APIs, authentication, dashboard modules, booking workflow, and management functionalities.

The system automates the complete booking lifecycle—from customer inquiry to photographer assignment, payment tracking, and project completion—making event management efficient and organized.

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/pancholipankaj3008/Weddings-Booking-Management-System.git
```

Install Frontend

```bash
cd CLIENT
npm install
npm run dev
```

Install Backend

```bash
cd SERVER
npm install
node server.js
```

---

# 🔑 Environment Variables
